## Installation

Install with dotnet from nuget

```bash
  dotnet add package isaaholic.ReadyWitted --version 1.0.0
```
    
## Usage/Examples

```csharp
namespace isaaholic.ReadyWitted 

ReadyWitted.CreateInstance("YOUR_API_KEY");
```
